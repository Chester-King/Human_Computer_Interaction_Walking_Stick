## Human_Computer_Interaction_Walking_Stick
Designing an advance Walking stick for the visually impaired population 
### Objective
The objective of this project is to design a walking stick which can *detect obstacles* on the path of the user and simultaneously keep track of the *location* i.e. latitudes and longitudes witht the time by using GPS
### Hardware Used
* Arduino Uno board
* Bread board
* Connecting wires
* Ultrasonic Sensors
* Buzzers
* esp8266 module
